### ⚙️ Project Description & My Task
My main objective for this final project was to bridge our design assets together into a fully interactive, clickable prototype. I connected the user journeys, homepage wireframes, and high-fidelity FreshRoots screens from the previous projects into a seamless, unified mobile application flow.

### 🛠️ Core Features Tested
* **Interactive Navigation**: Linked the wireframe elements to real user action paths, allowing clickable buttons to navigate across multiple application states.
* **Flow Validation**: Tested complete user entry from the initial structural stages straight through to the finalized brand-identity grocery shopping interfaces.
* **User Experience (UX) Review**: Verified that transitions between product sections, sliders, and page states feel natural, smooth, and intuitive for mobile app users.
